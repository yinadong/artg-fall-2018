Example 2: creating a scatterplot

Why

As discussed in class, a scatterplot is an effective exploratory visualization for examnining "correlation" between two variables. In this case, I wanted to look at what, if any, correlation exists between the level of economic development (as measured by GDP per capita) in a given country, and its infant mortality rate.

A few design and technical notes

  • Some basic styling is applied with the Bootstrap CSS library
  • Note the choice of scales: since GDP per capita (x-axis) varies greatly, a log scale is more appropriate. The same goes for the y-axis for infant mortality rate. You can experiment with different scale types.
  • The color and size of each entity is also significant--what do these visual attributes represent?
