Example 4: hierarchical layout

Why

Here we visualize the population count of people living with HIV, first by world regions and then by country. This is a hierarchical relationship.

A few design and technical notes

  • Note that to visualize hierarchical information, several visualization types are available. Please consider their pros and cons.
  • Another issue is that in order to aggregate data by world regions, we must combine data with metadata. See source code for how this works.
